About Tech Party Game

Tech Party Game is a small Django app built to make programming concepts stick through play. Quiz Mode tests fundamentals with a countdown; Bug Hunter asks you to spot what is broken in short code snippets; Who is Most Likely is a lighthearted party mode where every pick counts as a point — perfect for groups or solo smiles.

Admins can manage categories, questions, and browse saved scores from the Django admin panel. Have fun and keep learning!
